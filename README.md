# Student Portfolio Website

## 📌 Project Overview
This repository contains a well-structured student portfolio website built using core frontend technologies. It serves as a digital resume to showcase my background, highlight my projects, and demonstrate practical JavaScript application through a functional academic task planner.

## 📋 Course Requirements Fulfilled
*   **Home Page:** Inviting landing area introducing the site.
*   **About Me:** Details regarding my academic background and skills.
*   **Project Page:** Showcase grid of completed and ongoing technical works.
*   **Academic Planner:** Interactive tool for student task management.
*   **Contact Page:** Professional form for inquiries.

## 🛠️ Implemented Technical Features

### JavaScript Features
*   **Variables, Arrays & Objects:** Used for structured state and data management.
*   **Functions & Event Listeners:** Drives user interaction and application logic.
*   **DOM Manipulation:** Dynamically updates pages without full reloads.
*   **Local Storage:** Persists academic planner data across browser sessions.
*   **Form Validation:** Secures the contact form before submission.
*   **Responsive Navigation:** Hamburger menu for smooth mobile browsing.

### Responsive Design
The layout seamlessly adapts across all viewport breakpoints:
*   🖥️ Desktop
*   💻 Laptop
*   📱 Tablet
*   🔌 Mobile Phones

## 📂 Project Structure
```text
├── index.html        # Main markup file covering all required sections
├── index.css         # Main stylesheet for layout, themes, and responsiveness
└── index.js          # Core logic (Form validation, DOM manipulation, Planner)
```

## ⚙️ How the Academic Planner Works
1.  **Add Tasks:** Use the built-in form to create new items.
2.  **Complete Tasks:** Toggle task status visually.
3.  **Remove Tasks:** Clean up completed or unwanted items.
4.  **Data Persistence:** Tasks remain saved even if you refresh the browser.
